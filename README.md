这是HNU智能网联汽车实验的所有源代码
一共有四个实验，分别是：ROS部署，小车矩形行驶，小车巡线，小车识别交通标志并行驶

本仓库中的build、devel、src部署在文件夹catkin_ws下
其余代码与文件夹是训练分类标志模型所使用的代码、数据集以及训练结果。

具体实验步骤参见csdn对应博客
